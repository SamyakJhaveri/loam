# Technology Stack

> Conditional: loads on Python/build files. Loaded on demand to keep always-loaded context lean.

## Languages

- Python 3.12+ ‚Äî All pipeline code: harness, evaluation, augmentation, analysis, figure generation
- C/C++ (C++17) ‚Äî Benchmark kernel source (CUDA, OpenMP, OpenCL, SYCL targets)
- CUDA ‚Äî GPU kernel code in Rodinia, HeCBench, XSBench, RSBench, mixbench suites
- HTML/CSS/JavaScript ‚Äî Static dashboard in `visualizations/`
- LaTeX ‚Äî Paper in `docs/paper/latex/`
- Bash ‚Äî Batch runner scripts in `scripts/batch/`
- JSON ‚Äî Spec definitions (`specs/`), schemas (`schema/`), manifest (`manifest.jsonl`)

## Runtime

- Python 3.12.3 (Ubuntu 24.04)
- Virtual environment: `env_parbench/` (`source env_parbench/bin/activate`)
- Always `python3`, never bare `python`
- NVIDIA GeForce RTX 4070 (Ada Lovelace, sm_89, 12 GB VRAM)
- CUDA Toolkit 12.3 ‚Äî nvcc at `/opt/nvidia/hpc_sdk/Linux_x86_64/24.3/cuda/bin/nvcc`
- NVIDIA HPC SDK 24.3 ‚Äî nvc++ 24.3-0 for OpenMP target offloading
- OpenCL via NVIDIA CUDA ‚Äî headers/libs at `/opt/nvidia/hpc_sdk/Linux_x86_64/24.3/cuda/{include,lib64}`
- Intel oneAPI DPC++/C++ 2025.3.2 (SYCL, CPU-only ‚Äî no GPU backend)
- AMD Ryzen 9 7900X (12c/24t, 4.7 GHz base)
- GCC 12.4.0 / G++ 12.4.0 (Ubuntu)
- OpenMP via GCC `-fopenmp`

## Build Tools

- setuptools >= 68 ‚Äî Build backend (`pyproject.toml`)
- Package name: `parbench` v0.1.0
- Installable packages: `c_augmentation`, `harness`
- Optional dep groups: `[eval]`, `[analysis]`, `[dev]`, `[all]`
- nvcc 12.3 ‚Äî `/opt/nvidia/hpc_sdk/Linux_x86_64/24.3/cuda/bin/nvcc`
- nvc++ 24.3 ‚Äî NVIDIA HPC SDK (OpenMP target offload)
- gcc/g++ 12.4.0 ‚Äî OpenMP, serial C/C++
- icpx ‚Äî Intel oneAPI DPC++ (SYCL, CPU-only)
- Make ‚Äî Benchmark suites use Makefiles; harness executes `make` via spec build commands
- LaTeX toolchain ‚Äî `docs/paper/latex/Makefile`
- BibTeX ‚Äî `docs/paper/latex/references.bib`
- staticrypt ‚Äî npm package (CI only); AES-256 HTML encryption

## Configuration

- `config/paths.json` ‚Äî Machine-specific paths (project_root, downloads_root, hecbench_root)
- `config/paths.json.template` ‚Äî Template with `{{PROJECT_ROOT}}` placeholders
- Generated by `scripts/setup_claude_for_sam.sh`
- `schema/spec_schema.json` ‚Äî JSON Schema for Level 2 kernel specs
- `schema/manifest_schema.json` ‚Äî JSON Schema for manifest.jsonl
- `schema/reference_platform.json` ‚Äî Reference hardware/software platform
- `templates/spec_template.json` ‚Äî Template for new kernel specs
- `config/compiler_inventory.txt` ‚Äî Inventory of available compilers
- Build commands are spec-embedded (each spec JSON contains its own build commands)
- Compiler flags vary per spec (CUDA arch, OpenMP flags, OpenCL version targeting)
