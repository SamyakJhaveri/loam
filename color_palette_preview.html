Accessibility Check:
• Rose Pink (#C8607A) on white → contrast ratio 3.5:1 (meets WCAG AA for large text) — used for labels/headings only
• Saffron (#D48A35) on white → contrast ratio 3.3:1 — used for labels/headings only
• Teal (#2E8E9E) on white → contrast ratio 3.6:1 — used for labels/headings only
• Charcoal (#2D3436) on white → contrast ratio 14.5:1 — primary text, always accessible
• All colors have distinct luminance in grayscale: Rose=42%, Amber=58%, Teal=52%, Gold=68%, Charcoal=21%
• Colorblind-safe: Rose vs Teal provides hue + luminance contrast; Amber vs Teal provides hue contrast. No red-green only distinctions.
