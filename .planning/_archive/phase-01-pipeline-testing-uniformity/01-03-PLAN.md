   ```python
   """Analyze harness batch results for any suite and produce a results matrix.
   
   Reads individual .json files from results/harness/{suite}/ and produces:
     1. results/harness/{suite}/{suite}_results.json   -- structured JSON summary
     2. results/harness/{suite}/results_matrix_{suite}.md -- kernel x API matrix
   
   Usage:
       python3 scripts/analysis/analyze_harness_batch.py --suite rodinia
       python3 scripts/analysis/analyze_harness_batch.py --suite xsbench
   """
   ```

2. **Imports:** `from __future__ import annotations`, `argparse`, `json`, `sys`, `pathlib.Path`, `datetime.datetime`.

   ```python
   PROJECT_ROOT = Path("/home/samyak/Desktop/parbench_sam")
   APIS = ["cuda", "omp", "opencl"]
   ```

4. **`get_all_slugs(log_dir: Path, suite: str, apis: list[str]) -> list[str]`:**
   Generalized from Rodinia version. Globs for `{suite}-*.json`, splits on `-`, checks `parts[-1] in apis`, extracts slug as `"-".join(parts[1:-1])`. Returns sorted list.

5. **`load_json_result(log_dir: Path, suite: str, slug: str, api: str) -> dict | None`:**
   Reads `log_dir / f"{suite}-{slug}-{api}.json"`.

6. **`load_log_text(log_dir: Path, suite: str, slug: str, api: str) -> str`:**
   Reads `log_dir / f"{suite}-{slug}-{api}.log"`.

7. **`classify_result(log_dir: Path, suite: str, slug: str, api: str) -> dict`:**
   Same logic as Rodinia version, but uses `suite` parameter to construct `spec_id = f"{suite}-{slug}-{api}"`.

8. **`build_matrix(all_results: list[dict], suite: str) -> str`:**
   Same format as Rodinia version but with:
   - Title: `f"# {suite.title()} Full Batch -- Results Matrix"`
   - `**Total specs**` line uses actual count from results

9. **`main() -> int`:**
   - `argparse.ArgumentParser` with `--suite` (required, type=str) and `--project-root` (optional, default=PROJECT_ROOT).
   - Log directory: `project_root / "results" / "harness" / suite`
   - Output JSON: `log_dir / f"{suite}_results.json"`
   - Output matrix: `log_dir / f"results_matrix_{suite}.md"`
   - `log_dir.mkdir(parents=True, exist_ok=True)` before writing (Pitfall 3).
   - If log_dir has no JSON files, print warning and still write empty results.
   - Same flow as Rodinia version: get_all_slugs -> classify_result for each slug/api -> build_matrix -> write JSON + MD.

10. `if __name__ == "__main__": sys.exit(main())`
