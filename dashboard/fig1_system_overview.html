Figure 1. ParBench system architecture. Benchmark sources (Rodinia, HeCBench) are formalized into 185 kernel specifications through a validated JSON schema with a three-tier file security model. The harness engine provides automated build, run, and verify pipelines. Two experimental pathways leverage the harness: (1) AST-driven augmentation applies semantics-preserving code transforms at four difficulty levels, and (2) LLM evaluation tests code translation between parallel APIs with automated retry on failure. Results feed into analysis scripts and interactive dashboards.
