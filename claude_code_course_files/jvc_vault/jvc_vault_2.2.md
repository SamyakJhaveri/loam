# Workspace Blueprint (Full Template)

## What You'll Get From This 

The full production folder template with routing tables, context files, and workspace structure already built. Download it, open it in VS Code or point Claude at it, and start customizing.

## What This Is

This is the folder template from the Folder Architecture video in The Foundation (3.1). It runs on the three-layer routing system: a top-level CLAUDE.md that gives Claude identity and navigation, workspace-level context files that tell it what each area does, and skill files that plug in tools and processes where you need them.

The template uses a fake project with fake blog posts and demo scripts. That is intentional. Swap the names, rewrite the context files for your own work, and the structure holds. If you are a content creator, rename writing-room to script-lab. If you are a freelancer, swap community for client-intake. The layers stay the same.

## How to use it: 

Download the folder. Open it in VS Code or point Claude at it from your file system. Read (or have Claude read) the CLAUDE.md first to see how it routes. Then start editing the context files in each workspace to match what you actually do.

If you have not watched the Folder Architecture walkthrough yet, do that first. The template makes a lot more sense after you see it in action.
