Solved Problems (by uncommitted fixes)

FIXED
Fix V: _fatal_diagnostics threshold

Lowered from >= Fatal (4) to >= Error (3). Now catches real compile errors during validation.

FIXED
Fix A: Overlapping nested subscripts

Added overlap filter + merged re-validation in AstTransform.apply(). Prevents corruption from J[iS[i]*cols+j]-style expressions.

FIXED
Fix B: Struct member access precedence

arr[i].member now emits (*((...))) when . follows. Unit test: test_pointer_arithmetic_struct_member.

FIXED
Fix C: Assignment-in-condition swap

Added _contains_assignment() check. Skips SwapCondition when either operand contains = or compound assignment.

Unsolved Problems

BUG D
TypedefExpansion struct pointer breakage

Expands Point* to underlying struct type without handling pointer declarators. Breaks streamcluster-{cuda,omp} at L2+. Severity: HIGH.

FIXED
.cl file inconsistency

spec_loader.py now augments .cl files -- added to suffix list on Day 2 (2026-03-19). Both spec_loader.py and augment_verify.py are now consistent.

LOW
augment_verify.py PROJECT_ROOT bug

Path(__file__).parent.parent resolves to scripts/, not project root. Workaround: pass --project-root. run_augment_batch.py uses .parent.parent.parent (correct).

LOW
