# **What You'll Get From This**

Three complete, ready-to-use folder setups for three different types of work. Download one, replace the brackets, and you have a working system.

---

# **What This Is**

These are not just file trees. Each one is a complete folder setup: [CLAUDE.md](http://CLAUDE.md), workspace [CONTEXT.md](http://CONTEXT.md) files, naming conventions, routing tables, and process descriptions. Every field that you need to customize is in brackets with a description of what to write.

## **Three starter packs:**

#### **Content Pipeline**

For content creators. Three workspaces: Script Lab (writing and ideation), Production (building assets), Distribution (publishing and scheduling). Includes voice and style fields, platform-specific formatting rules, and a production process from script to output.

---

#### **Client Management**

For freelancers and consultants. Client template folder you duplicate per engagement. Templates workspace for reusable proposals and reports. Business dev workspace with pipeline stages and case study format. Confidentiality rules built into the [CLAUDE.md](http://CLAUDE.md).

---

#### **Code Project**

For developers. Four workspaces: Planning (specs, architecture, decisions), Src (application code), Docs (API docs, guides, changelog), Ops (deploy, monitoring, scripts). Includes spec template, decision record template, and context files for code conventions and testing.

---

#### **How to use them:**

1. Download the one closest to your work
2. Create the folder structure on your computer
3. Copy the [CLAUDE.md](http://CLAUDE.md) and [CONTEXT.md](http://CONTEXT.md) files into their folders
4. Replace every `[bracketed field]` with your information
5. Point Claude Code at the root folder and start working
6. Edit the context files as you learn what Claude needs to know

If none of these match your work exactly, they still give you the structure. Edit the workspace names, change the routing table, and rewrite the context files. The three-layer pattern (map → rooms → tools) holds regardless of what you do.
