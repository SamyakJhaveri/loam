## How The Vault Works
You are in the resource library. This is where everything I use, recommend, and build with lives. Templates, reference docs, production examples, frameworks, and structured courses. All of it gets updated regularly.

### Three things to know before you dive in:
1. The docs are not textbooks. 

You do not need to read them front to back. They are designed to be dropped straight into Claude. Start a conversation, upload the doc, and tell Claude what you want to build or learn. It will walk you through it step by step, answer your questions, and help you build off of it. This works whether you are a developer or you have never written a line of code in your life.

2. New resources get added every month. 

New tools I test, new templates I build, new links worth saving. What you see in here today will look different a month from now. Check back often. If there is something you want covered that you do not see yet, post about it in the community and I will prioritize it.

3. This is also a course space. 

Below the library, you will find two structured courses. These go deeper than the free content. The free course (The Foundation) teaches the ideas and frameworks. These courses teach you how to apply them to real work with real examples and guided builds.

## Where to start
If you just joined Premium, here is the order I would go in:

1. Set up Discord (next page) so you have real-time access to help

2. Download the Workspace Blueprint if you want the full production folder template

3. Browse the library and grab whatever is relevant to what you are building right now

4. Start a course when you are ready to go deeper

If you have been in the free course for a while and already have your folder set up, jump straight to the Production CLAUDE.md Examples or the Workflow Templates. Those will level up what you already built.
