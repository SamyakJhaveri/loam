Key Design Decisions

Dev Experience

.claude/ at repo root is a symlink to seed/.claude/

Claude Code works normally in the template repo. Edits to .claude/ are edits to seed/.claude/. One source of truth.

CLAUDE.md Split

Root CLAUDE.md = template dev docs
seed/CLAUDE.md.jinja = project template

No more TEMPLATE-CLAUDE.md confusion. Each file describes its own context.

Soil Metaphor

seed → planted in projects
cultivation → skills grow here
soil → knowledge base

Natural naming aligned with "Loam" identity. Replaces cryptic _research, seed-skills, claude_code_course_files.

Jinja Files Reduced

14 .jinja → ~7 essential templates

ARCHITECTURE.md, DESIGN.md, PROJECT-BACKGROUND.md, SCRIPTS_DIRECTORY.md, CONTEXT.md can be generated by skills on demand rather than shipped as templates.
