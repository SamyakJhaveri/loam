# academic-research

4-skill academic pipeline — deep research, paper writing, peer review, pipeline orchestrator.

**Upstream:** https://github.com/imbad0202/academic-research-skills

## Install

```bash
/plugin marketplace add Imbad0202/academic-research-skills
```

Requires Claude Code v3.7.0+.

## Skills

- **deep-research** — 13-agent systematic review with PRISMA support
- **academic-paper** — 12-agent paper writing with LaTeX/DOCX output
- **academic-paper-reviewer** — 7-agent peer review with 0-100 rubrics
- **academic-pipeline** — 10-stage orchestrator with integrity gates

## Slash Commands

`/ars-research`, `/ars-paper`, `/ars-review`, `/ars-pipeline`, `/ars-search`, `/ars-cite`, `/ars-outline`, `/ars-revise`, `/ars-export`, `/ars-status`
