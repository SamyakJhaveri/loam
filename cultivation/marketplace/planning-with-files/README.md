# planning-with-files

Manus-style persistent markdown planning. Creates `task_plan.md`, `findings.md`, `progress.md` for cross-session persistence with hook-based re-injection after `/compact`.

**Upstream:** https://github.com/othmanadi/planning-with-files

## Install

```bash
npx skills add planning-with-files
```

## Overlap Warning

Heavily overlaps with Loam's built-in planning (feature-dev, plan mode, handoff, rules, memory). Install only if you prefer file-based planning over Loam's approach.

## Requirements

- Node.js
- Bash or PowerShell
