# understand-anything

Multi-agent codebase analysis pipeline with interactive React knowledge graph dashboard.

**Upstream:** https://github.com/Lum1104/Understand-Anything

## Install

```bash
/plugin marketplace add Lum1104/Understand-Anything
```

## Commands

`/understand`, `/understand-dashboard`, `/understand-chat`, `/understand-diff`, `/understand-domain`

## Overlap Warning

Overlaps with Semble + CodeGraphContext. Unique value is the interactive React dashboard for visual architecture exploration. Best for onboarding new team members.
